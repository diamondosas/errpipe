Here are focused snippets for each concept:

---

**1. The Engine — `exec.Command()`**
```go
cmd := exec.Command("ls", "-la")
cmd.Run()
```

---

**2. The Success Path — Pipe stdout to screen**
```go
cmd := exec.Command("echo", "Hello, World!")
cmd.Stdout = os.Stdout
cmd.Run()
```

---

**3. The File Prep — `os.OpenFile()` with append + create**
```go
file, err := os.OpenFile("bug_report.txt", os.O_APPEND|os.O_CREATE|os.O_WRONLY, 0644)
if err != nil {
    log.Fatal(err)
}
defer file.Close()
```
The three flags do the following:
- `O_APPEND` — writes go to the end of the file
- `O_CREATE` — creates the file if it doesn't exist
- `O_WRONLY` — opens in write-only mode

---

**4. The Magic Splitter — `io.MultiWriter`**
```go
multi := io.MultiWriter(file, os.Stderr)

// Anything written to `multi` now goes to BOTH targets simultaneously
multi.Write([]byte("this goes to the file AND the screen\n"))
```

---

**5. The Execution — Wiring it all together**
```go
cmd := exec.Command("ls", "/nonexistent") // a command that will error

cmd.Stdout = os.Stdout      // success output → screen
cmd.Stderr = multi          // error output   → screen + file

cmd.Run()                   // output streams live as the process runs
```

---

**Full working example**
```go
package main

import (
    "io"
    "log"
    "os"
    "os/exec"
)

func main() {
    // Prep the file
    file, err := os.OpenFile("bug_report.txt", os.O_APPEND|os.O_CREATE|os.O_WRONLY, 0644)
    if err != nil {
        log.Fatal(err)
    }
    defer file.Close()

    // Build the splitter
    multi := io.MultiWriter(file, os.Stderr)

    // Wire up the command
    cmd := exec.Command("ls", "/nonexistent")
    cmd.Stdout = os.Stdout
    cmd.Stderr = multi

    // Run it
    cmd.Run()
}
```

Run this and you'll see the error printed to your terminal **and** saved to `bug_report.txt` at the same time.
